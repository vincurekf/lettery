lettery

lettery is simple jquery plugint that finds specified letter and makes edit to it.

Usage

lettery accepts two parametters: letters/words and options. 

$( '.lettery' ).lettery( letters, options );

letters/words should be an array of letters or words, so for one letter it would be: [ "w" ], for more letters: [ "w", "t", "f" ],
and for words: [ "bone", "ribs" ].
If you want to use words, you neet do define option words to true

options: styles can be any css: 

styles: {
  'position': 'relative',
  'color': '#f00'
},

options: class if you wish use custom css class for single letter, define that class name here

Full example: 

$( '.lettery' ).lettery( [ "bone", "ribs", "bacon", "chicken", "pork" ], {
  styles: {
    'position': 'relative',
    'bottom': '4px',
    'color': '#e46'
  },
  class: "word",
  words: true
});
