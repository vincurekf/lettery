lettery

lettery is simple jquery plugint that finds specified letter and makes edit to it.

Usage

lettery accepts two parametters: letters and options. 

$( '.lettery' ).lettery( letters, options );

letters should be an array of letters, so for one letter it would be: [ "w" ], for more letters: [ "w", "t", "f" ].
options: styles can be any css: 

styles: {
  'position': 'relative',
  'color': '#f00'
},

options: class if you wish use custom css class for single letter, define that class name here

Full example: 

$( '.lettery' ).lettery( [ "w", "t", "f" ], {
  styles: {
    'position': 'relative',
    'bottom': '4px',
    'color': '#e46'
  },
  class: "letter"
});
